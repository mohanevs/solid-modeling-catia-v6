# 🧱 Solid Modeling with CATIA V6 & DELMIA V5

This repository showcases a collection of 3D mechanical models, assemblies, and drawings created using **CATIA V6** and **DELMIA V5**. The models are based on standard components commonly found in engineering drawing references and real-world mechanical systems.

## 📁 Folder Structure

### 🔹 `3D Models/`

* Contains **57 individual 3D part files** (.CATPart).
* Models are inspired by standard mechanical components from engineering drawing textbooks.
* Each model was independently designed and rendered in CATIA.

### 🔹 `Assembly/`

* Includes **7 full mechanical assemblies**, each in its own folder:

  * Screw Jack
  * Knuckle Joint
  * Pipe Vice
  * Plummer Block
  * Press Tool
  * Tool Head of Shaper
  * Motor Blower
* Each folder includes:

  * The assembled CATIA product file (.CATProduct)
  * The individual part files used in the assembly

### 🔹 `Drawing/`

* Contains 2D drawing sheets of selected 3D models in **PDF format**.
* Each drawing is plotted on a standard title block with dimensioning and annotations.
* **Note:** Personal identifiers (name and roll number) have been redacted for privacy.

### 🔹 `SM Application/`

* Features **8 3D models of real-world objects**, including:

  * Boat
  * Classic Plane
  * Hex Nut
  * Robotic Arm
  * Star
  * Sphere
  * Triangular Bracket
  * Square Thread
* These models are more creative or practical in nature, showcasing real-world applications of solid modeling techniques.

## 🛠️ Tools Used

* **CATIA V6** – For 3D modeling and assemblies
* **DELMIA V5** – For manufacturing process simulation (where applicable)
